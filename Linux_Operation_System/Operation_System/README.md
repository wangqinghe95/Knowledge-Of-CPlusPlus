## Process And Thread
### Thread
#### 什么是线程（thread）？C++ 中如何创建和管理线程？
+ 线程是操作系统能够进行运算调度的最小单位，它被包含在进程中，是进程中的实际运行单位。一个线程是进程中一个单一顺序的控制流，一个进程可以并发多个线程执行不同的任务。
